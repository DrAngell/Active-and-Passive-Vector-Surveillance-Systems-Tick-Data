# Active-and-Passive-Vector-Surveillance-Systems-Tick-Data
Dataset for "Active and Passive Vector Surveillance Systems in Lyme Endemic Minnesota and Their Correlation to Human Disease", 2025 (JPHMP)

This is a cleaned dataset providing verified observation counts of Ixodes, Dermacentor, and all ticks in Minnesota from 2018-23. The original data was sourced from the iNaturalist open access portal. The dataset provides an example of transformed crowd sourced passive surveillance information used in conventional epidemiologic analysis re: One Health. This dataset was used in the analysis of published manuscript "Active and Passive Vector Surveillance Systems in Lyme Endemic Minnesota and Their Correlation to Human Disease", 2025 (JPHMP).
